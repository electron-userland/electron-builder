:::tip[Upgrading from v26?]
Many configuration keys were renamed or restructured in v27 (e.g. `mac.sign`, `win.sign`, `electronGet`, `asar`, `nativeModules`, `snapcraft`). Run **`electron-builder migrate-schema`** to rewrite your config automatically, then skim [What's New in v27](/docs/migration/whats-new-v27) and the full [v27 Breaking Changes](/docs/migration/v27-breaking-changes).
:::
